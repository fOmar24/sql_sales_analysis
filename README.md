📊 SQL Sales Analysis
📌 Project Overview

This project analyzes sales data using SQL to uncover insights about revenue trends, customer behavior, and product performance. The goal is to demonstrate data analysis techniques using structured queries for business decision-making.

The analysis focuses on:

Sales performance

Customer purchasing behavior

Product profitability

Revenue trends

🗂 Dataset

The dataset contains information about:

| Column      | Description                      |
| ----------- | -------------------------------- |
| order_id    | Unique identifier for each order |
| order_date  | Date the order was placed        |
| customer_id | Unique customer identifier       |
| product_id  | Unique product identifier        |
| quantity    | Number of products purchased     |
| price       | Price per unit                   |
| total_sales | Total revenue from the order     |


Project Objectives

The analysis answers key business questions such as:

What are the total sales revenue?

Which products generate the most revenue?

Who are the top customers by spending?

What are the monthly sales trends?

Which products sell the highest quantity?
